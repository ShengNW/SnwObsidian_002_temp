# 完整学生续居留材料包（打印勾选版）

> 目标：去省政府或分省政府现场催办 `attestation de prolongation d'instruction` 时，按“完整学生续居留材料包”携带。  
> 建议整理方式：`1 份原件 + 1 套复印件 + 手机里保存 PDF/截图`

## 0. 出发前整理

- [ ] 所有材料按类别分成独立小包
- [ ] 原件单独放一袋
- [ ] 复印件单独放一袋
- [ ] 手机里保存所有 PDF、截图、邮件和网页记录

## 1. 身份与案件材料（必须带）

- [ ] 护照原件
- [ ] 护照相关页复印件（身份页、签证页、重要出入境页）
- [ ] 当前或最近一次 `titre de sejour` / `VLS-TS`
- [ ] 已过期的 `attestation de prolongation d'instruction`
- [ ] `ANEF` 递交证明
- [ ] `ANEF` 案件编号
- [ ] `ANEF` 进度页面截图
- [ ] 发给 `Thionville` 的邮件正文打印件
- [ ] 对方自动回复打印件
- [ ] 近期证件照 / `e-photo`

## 2. 学校材料（一定带）

- [ ] `certificat d'inscription` 或 `certificat de reinscription`
- [ ] 上一学年的全部成绩单
- [ ] 法国最近取得的 `diplome`（如有）
- [ ] `attestation de reussite`（如有）

## 3. 资金材料（按实际情况带全）

- [ ] 个人银行余额证明 / `solde crediteur suffisant`
- [ ] 资助人定期转账证明（如适用）
- [ ] 资助声明（如适用）
- [ ] 资助人身份证明复印件（如适用）
- [ ] 奖学金证明（如适用）
- [ ] 最近 3 个月工资单（如适用）

## 4. 住址与住房材料（按居住方式准备）

> 下面三种情况按你的实际住法选择；核心目标是准备好 1 份近 6 个月内的住址证明。

### 自己租房

- [ ] `bail` 租房合同
- [ ] 最近一张 `quittance de loyer`
- [ ] 最近的电 / 气 / 水 / internet / 电话账单之一（如有）

### 住在别人家

- [ ] `attestation d'hebergement`
- [ ] 接待人身份证或居留卡复印件
- [ ] 接待人的住址证明

### 住酒店

- [ ] 酒店证明或上月发票

## 5. 你这次特别不要漏的材料

- [ ] 学校证明
- [ ] 成绩单
- [ ] 资金证明
- [ ] 住房合同（`bail`）
- [ ] 房租收据（`quittance de loyer`）
- [ ] 住房 / 接待证明（如适用）
- [ ] 签证 / 护照
- [ ] 之前的 `prolongation`
- [ ] 之前的 `VLS-TS` / 旧卡

## 6. 现场核心说辞（可选打印）

`J'ai deja depose un dossier complet sur l'ANEF. Mon attestation de prolongation d'instruction est expiree. Je demande son renouvellement immediat ou une solution provisoire.`
